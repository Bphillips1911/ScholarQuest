# PBIS Teacher-Student Privacy Protection System

## How Teachers Add Students to Their PBIS Classroom

### Privacy-First Approach

**1. Role-Based Access Control**
- 6th Grade Teachers: Can only see 6th grade students
- 7th Grade Teachers: Can only see 7th grade students  
- 8th Grade Teachers: Can only see 8th grade students
- Unified Arts Teachers: Can see all 6-8 grade students (for PE, Art, Music, etc.)
- Administration/Counselors: Can see all students for oversight

**2. Student Information Protection**
- Teachers only see: Student Name, Student ID (like BH6001), House Assignment, Current Points
- Teachers CANNOT see: Home addresses, phone numbers, parent information, medical records
- All sensitive data is stored separately and only accessible to administration

**3. Secure Student Lookup System**
Teachers add students using one of these secure methods:

**Method 1: Student ID Search**
- Teacher enters official Student ID (e.g., "BH6001")
- System verifies teacher has permission for that grade level
- Only shows students the teacher is authorized to see

**Method 2: Grade-Level Student Lists**
- Teacher selects their authorized grade level
- System shows only students in grades they can access
- Teacher selects from pre-approved list

**Method 3: House-Based Assignment**
- Teacher can see students by House within their grade permissions
- Helps with house competitions and activities

### Technical Privacy Safeguards

**Database Security:**
- Student records are encrypted
- Access logs track all teacher interactions
- Teachers cannot export or download student data
- Session timeouts prevent unauthorized access

**Permission Validation:**
- Every teacher action is checked against their role permissions
- Cross-grade access attempts are blocked and logged
- Admin oversight of all teacher-student assignments

**Data Minimization:**
- Teachers only see data necessary for PBIS activities
- Personal/family information is completely separate
- MUSTANG point awards and behavioral notes only

### Parent Notification System
- Parents receive notifications when their child is added to a teacher's PBIS classroom
- Parents can view which teachers have access to their child's PBIS data
- Parents can request teacher assignment changes through administration

This system ensures teachers can effectively manage their PBIS classroom while maintaining strict privacy protection for student information.
